# How to use it on Termux?
**Installing Git, Unzip, Node.js LTS, and more..**
```bash
pkg update -y && pkg upgrade -y
pkg install git -y
pkg install nodejs-lts -y
pkg install unzip -y
pkg install jp2a -y
```
**Cloning Repository**
```bash
git clone https://github.com/NekomonHub/neuroobfuscator
cd neuroobfuscator
unzip neuroobfuscator.zip
```
**Start Tool**
```bash
node nayla.js
```
**Examples of its use**
```bash
node nayla.js yourfiles.js output.js
```

> - yourfiles.js *is the name of the file you want to obfuscate*
> - output.js *Is the file name for your obfuscator output.*
